优点：
1. “资源高内聚” 组件资源内部高内聚，组件资源由自身加载和控制
2. “作用域独立” 内部结构密封，不与全局和其他组件产生影响
3. “自定义标签” 专门用来定义组件 angular directive
4. “可相互结合” 可随意挑选组件库中的任意几个组件构成一个独立的模块，需要自己根据UI设计来布局进行组装插件
